# Assignment_Webmapping

## Project Overview

**This project is a web mapping application that visualizes bus stops in Unterschleißheim, a town in Bavaria, Germany, located about 17 km north of Munich. Using Leaflet 1.9.4, the map displays bus stops for individuals traveling from Unterschleißheim Ost toward the west.

The purpose of this project  is to practice working with Leaflet, GeoJSON, and client-side JavaScript for interactive maps.
